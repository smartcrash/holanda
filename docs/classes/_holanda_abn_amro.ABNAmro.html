requestAccessToken

  • requestAccessToken(options): Promise<RequestAccessTokenResponse>

  • Retrieves an access token and refresh Token. The access token and refresh token are as used to ensure that authentication is complete and that consent has been given by the client to the consumer. The access token is used when calling APIs on the ABN AMRO API gateway.

    Parameters

    • options: RequestAccessTokenOptions

    Returns Promise<RequestAccessTokenResponse>

    Example

    const response = await client.requestAccessToken({ grantType: 'client_credentials' })
    // {
    //   access_token: '3srrbf0WcPTMKCwGBWgXdU6mNnoj',
    //   token_type: 'Bearer',
    //   expires_in: 7200
    // }

    See

    https://developer.abnamro.com/api-products/authorization-code#tag/Access-and-refresh-token/operation/requestAccessToken

    • Defined in ABNAmro.ts:92

requestAuthCode

  • requestAuthCode(options): string

  • Retrieves an authorization code. The authorization code is used to ensure that an ABN AMRO client or end user has provided permission to the API consumer. This authorization code is passed to the authorization server when requesting and access and a refresh token.

    Parameters

    • options: RequestAuthCodeOptions

    Returns string

    Example

    const response = client.requestAuthCode({
      scope: [ABNAmroScopes.PostSEPAPayment],
      redirectUri: 'https://localhost/auth',
      responseType: 'code',
    })
    // https://auth-sandbox.abnamro.com/as/authorization.oauth2?client_id=TPP_test&response_type=code&scope=psd2%3Apayment%3Asepa%3Awrite&redirect_uri=https%3A%2F%2Flocalhost%2Fauth
